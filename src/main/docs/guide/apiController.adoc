For the Announcement API Controller create we use the http://docs.grails.org/latest/ref/Command%20Line/create-controller.html[create-controller command]
[source, bash]
----
./grailsw create-controller ApiAnnouncement
Created grails-app/controllers/grails/test/security/ApiAnnouncementController.groovy
| Created src/test/groovy/grails/test/security/ApiAnnouncementControllerSpec.groovy
----

We need to add the next line to the mappings Closure.
